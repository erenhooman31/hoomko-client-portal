# پرتال مشتریان و مدیریت پروژه

یک نمونه کار full-stack UI برای نمایش پرتال مشتریان، مدیریت پروژه، تیکت، بودجه، فازهای تحویل و وضعیت پشتیبانی.

## لینک ها

- نسخه آنلاین: https://hoomko-client-portal.vercel.app
- مخزن GitHub: https://github.com/erenhooman31/hoomko-client-portal

## قابلیت ها

- داشبورد فارسی و RTL برای پروژه ها، تیکت ها و وضعیت تحویل
- صفحات کامل برای داشبورد، پروژه ها، تیکت ها، صورت حساب و فایل ها
- انتخاب پروژه و مشاهده فناوری، بودجه، وضعیت و درصد پیشرفت
- فیلتر تیکت ها بر اساس وضعیت
- تایم لاین تحویل مرحله ای پروژه
- پیاده سازی با React و Vite به عنوان نمونه فرانت اند قابل توسعه

## هدف نمونه کار

این پروژه برای نمایش توانایی در طراحی پنل مدیریتی، ساخت پرتال مشتریان، جریان پشتیبانی و مدیریت تحویل مرحله ای پروژه های وب و اتوماسیون ساخته شده است.

## اجرای محلی

```bash
npm install
npm run dev
```
